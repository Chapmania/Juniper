Detailed Description

Implementations for each of the values in TweenType. All functions receive a value t, representing the proportion of the way through the animation process we are calculating. All functions take an extra k parameter, for scaling tertiary effects; most functions do not use it. All functions return a value v, mapping the value t to one of the desired 'tween shapes.
