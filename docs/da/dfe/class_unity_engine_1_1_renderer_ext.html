bool UnityEngine.RendererExt.UseShared
static
Initial value:
=>
Application.isEditor && !Application.isPlaying

Returns true if we are running in the editor, but the player is not running. In these cases, we want to set sharedMaterials instead of materials on objects to avoid creating duplicate objects that will leak into memory.
