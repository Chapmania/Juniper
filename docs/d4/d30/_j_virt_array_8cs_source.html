6 /*
7  * About virtual array management:
8  *
9  * Full-image-sized buffers are handled as "virtual" arrays. The array is still accessed a strip at a
10  * time, but the memory manager must save the whole array for repeated
11  * accesses.
12  *
13  * The Access method is responsible for making a specific strip area accessible.
14  */
