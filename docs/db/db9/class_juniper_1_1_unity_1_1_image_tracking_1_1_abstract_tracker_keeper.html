inlineprivate

Creates an event handler that can handle losing a given target. The event handler will fire the targetChanged event if the lost target was the CurrentTarget. If losing this target means that there are no tracked targets, then the onAllTargetsLost event is fired. Finally, Vuforia's Persistent Extended Tracking is reset, if it was active.
