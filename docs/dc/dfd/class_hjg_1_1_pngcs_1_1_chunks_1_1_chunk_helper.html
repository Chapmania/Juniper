inlinestatic

Ad-hoc criteria for 'equivalent' chunks.

Two chunks are equivalent if they have the same Id AND either:

  1. they are Single
  2. both are textual and have the same key
  3. both are SPLT and have the same palette name Bear in mind that this is an ad-hoc, non-standard, nor required (nor wrong) criterion. Use it only if you find it useful. Notice that PNG allows to have repeated textual keys with same keys.
