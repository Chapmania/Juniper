Insert an item at an arbitrary point in the buffer.

Parameters
indexThe index at which to insert item .
itemThe value to insert.
Exceptions
ArgumentOutOfRangeExceptionIf index is less than 0, or greater than Count - 1.

If the buffer is full, the oldest item will be dropped before the insertion is made. It's somewhat difficult to intuit the results, especially when inserting at index 0. The buffer always maintains that buffer.Insert(n, x); buffer[n] == x;, so inserting at index 0 when the buffer is full looks like overwriting index 0.

var buffer = new RingBuffer\<int\>(4);
buffer.Insert(0, 5); // { 5 }
buffer.Insert(0, 7); // { 7, 5 }
buffer.Insert(1, 3); // { 7, 3, 5 }
buffer.Insert(1, 13); // { 7, 13, 3, 5 }
buffer.Insert(2, 2); // { 13, 3, 2, 5 }
buffer.Insert(0, 17); // { 17, 3, 2, 5 }
