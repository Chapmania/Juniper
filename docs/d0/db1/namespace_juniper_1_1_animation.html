◆ TweenType

enum Juniper.Animation.TweenType
strong

Different modes of animating a single value from 0 to 1.

Enumerator
Linear 

The most basic form of `tween, values increase in proportion to time, from 0 to 1 in the forward direction, then 1 to 0 in the reverse direction.

LinearContinuous 

Values increase in proportion to time from -1 to 0 in the forward direction, then 0 to 1 in the reverse, rather than decreasing from 1 to 0.

Quadratic 

Values increase in proportion to the square of time, from 0 to 1 in the forward direction, then 1 to 0 in the reverse direction.

QuadraticContinuous 

Values increase in proportion to the square of time. from -1 to 0 in the forward direction, then 0 to 1 in the reverse direction.

Sine 
SineContinuous 
Bump 

Similar to Sine, but values bump in the opposite direction slightly, before the beginning and the end of the curve.

BumpContinuous 

Similar to Sine, but values bump in the opposite direction slightly, before the beginning and the end of the curve.
