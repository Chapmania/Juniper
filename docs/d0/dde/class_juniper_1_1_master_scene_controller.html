void Juniper.MasterSceneController.SwitchToSceneName ( string  sceneName,
bool  skipFadeOut = false 
)

A subscene is a root game object loaded from another scene. The scenes all get loaded at runtime and then you can make different parts of it visible on the fly. This procedure deactivates any subscenes that are not the desired subscene, calling any Exit functions along the way. In the new scene, TransitionController Enter functions are called as well. It is suitable for running in a coroutine to track when the end of the switching process occurs.
