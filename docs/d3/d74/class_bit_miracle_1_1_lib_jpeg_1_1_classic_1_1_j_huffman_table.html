Gets or sets a value indicating whether the table has been output to file.

It's initialized false when the table is created, and set true when it's been output to the file. You could suppress output of a table by setting this to true.

This property is used only during compression. It's initialized false when the table is created, and set true when it's been output to the file. You could suppress output of a table by setting this to true. (See jpeg_suppress_tables for an example.)
