◆ TweenType

enum Juniper.Unity.Animation.TweenType
strong

Different modes of animating a single value from 0 to 1.

Enumerator
Linear 

The most basic form of `tween, values increase in proportion to time.

Quadratic 

Values increase in proportion to the square of time.

Cubic 

Values increase in proportion to the cube of time. This function is specifically scaled to provide the full shape of the cube function from -1 to 1.

Sine 

Values increase along a sine wave (actually, 1 minus cosine of time scaled to π)

Bump 

Similar to Sine, but values bump in the opposite direction slightly, before the beginning and the end of the curve.

PingPong 

An inverted curve that plays both in and out.
