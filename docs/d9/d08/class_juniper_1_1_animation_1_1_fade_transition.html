override void Juniper.Animation.FadeTransition.OnEntered ( )
inlineprotectedvirtual

Saves the last clear color, culling mask, and ambient light mode, then prepares the camera to render the special "faded out" view, which allows TransparentFX objects to still be visible, so we can have simple UI elements still visible while the view is faded out. If Unity's audio subsystem is available, switches the audio source to use the fade-in sound for the next transition. Also, executes and queued actions that are waiting for the fade transition to complete.
