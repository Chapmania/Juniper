◆ audioSources

List<AudioSource> Juniper.Unity.Audio.InteractionAudio.audioSources
private

A pool of audio sources that can be moved around to play sounds wherever they are needed. The Play(Interaction, Transform, AbstractHapticDevice, Action) method will find whichever one is currently not playing and use it, or create a new one if all audio sources are currently playing. It starts out with 5 audio sources by default so that applications don't have to grow the collection right away.
