13  /// <summary>

15  /// and then you can make different parts of it visible on the fly. This procedure deactivates
16  /// any subscenes that are not the desired subscene, calling any Exit functions along the way. In
17  /// the new scene, TransitionController Enter functions are called as well. It is suitable for

341  private Task Stopper(Action<IProgress> actor, IProgress prog)
342  {
