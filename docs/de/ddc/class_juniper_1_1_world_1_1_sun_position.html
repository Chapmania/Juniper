Detailed Description

Calculates the position of the Sun in relation to the user's current position on spaceship Earth. Only one of this component is allowed on a gameObject at a time. This component also requires a GPSLocation component on the same gameObject and a CompassRose component somewhere in the scene. This component executes in the editor.
