Detailed Description

Represents an angle in degrees that doesn't wrap around to 0 when iterating past 360 degrees. It does this by making the update the smallest delta of value plus whole rotations. e.g. if the current value is 359, setting the value to 1 will actually return as 361, as Abs(361 - 351) < Abs(1 - 351).
