This class is used by both the compression and decompression code.

virtual void BitMiracle.LibJpeg.Classic.JpegErrorMessage.EmitMessage ( int  msg_level)
inlinevirtual

Conditionally emit a trace or warning message.

Parameters
msg_levelThe message severity level.
Values are:
-1: recoverable corrupt-data warning, may want to abort.
0: important advisory messages (always display to user).
1: first level of tracing detail.
2, 3, ...: successively more detailed tracing messages.

The main reason for overriding this method would be to abort on warnings. This method calls output_message for message showing.

An application might override this method if it wanted to abort on warnings or change the policy about which messages to display.

int BitMiracle.LibJpeg.Classic.JpegErrorMessage.NumWarnings => m_numWarnings

Gets the number of corrupt-data warnings.

The num_warnings.

For recoverable corrupt-data errors, we emit a warning message, but keep going unless emit_message chooses to abort. emit_message should count warnings in Num_warnings. The surrounding application can check for bad data by seeing if Num_warnings is nonzero at the end of processing.
