struct  Angle
 Represents an angle in degrees that doesn't wrap around to 0 when iterating past 360 degrees. It does this by making the update the smallest delta of value plus whole rotations. e.g. if the current value is 359, setting the value to 1 will actually return as 361, as Abs(361 - 351) < Abs(1 - 351). Nits 

Candelas per Square Meter, the SI unit of Luminance, a measure of light emitted per unit area. Conversions between Lumens and Nits are not accurate, but useful for working between ARCore and Magic Leap.
