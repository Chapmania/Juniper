Detailed Description

Sorts a collection of strings, by first splitting out any numbers and treating them as numbers, rather than lexicographic strings.

var files = new List<string> { "file003", "file1", "file04", "file02" }; /// files.Sort(new NatrualSortComparer()); /// files.Matches(new []{ "file1", "file02", "file003", "file04" }); // --> true
